# python-password-generator
This is a simple Password Generator project built using Python.

The program generates a random password according to the length chosen by the user. It uses:
- Small letters
- Capital letters
- Numbers
- Special characters

to create a strong and random password.

## Features

- User-defined password length
- Random password generation
- Uses letters, numbers, and symbols
- Beginner-friendly Python project
- Simple and clean logic

## Technologies Used

- Python

## Concepts Used

- Variables
- Strings
- Input/Output
- For Loop
- Random Module
- String Concatenation

## How It Works

1. User enters password length
2. Program combines all characters
3. Random characters are selected
4. Final password is generated and displayed
